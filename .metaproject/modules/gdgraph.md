# gdgraph

## Purpose

Builds code graph, symbol graph, dependency map, and affected context.

## Commands

- `gd-metapro gdgraph build`
- `gd-metapro gdgraph query "<query>"`
- `gd-metapro gdgraph affected <target>`
- `gd-metapro gdgraph explain <target>`
