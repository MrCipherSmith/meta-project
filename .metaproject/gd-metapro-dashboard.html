Agent Workflow

1. Route

Start from index.md and select the module or skill that owns the question.

2. Navigate

Use gdgraph for related files, affected context, cycles, and module boundaries.

3. Compress

Use gdctx before loading large search output, diffs, command logs, or long files.

4. Verify

Read testing and health reports before claiming quality or gate status.

5. Learn

Write decisions and lessons to wiki, memory, and project skills when patterns change.
